Генератор паролей (Password Generator)

Краткое описание

Мобильное приложение для генерации безопасных паролей с настройкой сложности и возможностью сохранения истории.

Основные функции
Базовые функции (MVP)
Генерация случайных паролей

Настройка длины пароля (6-32 символа)

Выбор типов символов: прописные/строчные буквы, цифры, спецсимволы

Копирование пароля в буфер обмена

Индикатор сложности пароля

Дополнительные функции
История сгенерированных паролей

Темная/светлая тема

Проверка стойкости пароля

Навигация
Главный экран → кнопка "История" → Экран истории

Экран истории → кнопка "Назад" → Главный экран

Сценарии использования
Сценарий 1: Генерация пароля
Пользователь открывает приложение (главный экран)

Настраивает параметры (длина, типы символов)

Нажимает "Сгенерировать"

Приложение показывает пароль и его сложность

Пользователь нажимает "📋" для копирования

Сценарий 2: Просмотр истории
На главном экране пользователь нажимает "История"

Приложение показывает список ранее сгенерированных паролей

Пользователь может скопировать или удалить любой пароль

Распределение по лабораторным работам
ЛР4 (Верстка)

Базовая стилизация и адаптивный дизайн

ЛР5 (Логика)
Алгоритм генерации паролей

Логика расчета сложности пароля

Работа с буфером обмена

Управление состоянием интерфейса

ЛР6 (Данные)
Сохранение истории паролей в SharedPreferences

Интеграция с API для проверки утечек паролей

Локальное хранение настроек приложения

Работа с API
API для проверки утечек паролей:
Использование Have I Been Pwned API для проверки, не был ли пароль скомпрометирован в известных утечках данных.

Технические детали
Хранение данных
Настройки: SharedPreferences

История паролей: SQLite или SharedPreferences

Безопасность: пароли не сохраняются в открытом виде

Цветовая схема
Основной цвет: синий (#2196F3)

Фон: светлый/темный серый

Акценты: зеленый (сильный пароль), оранжевый (средний), красный (слабый)
